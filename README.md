# WATonomous Wiki website

The [website](https://wiki.watonomous.ca) for the WATonomous Wiki.

### Getting started

1. Clone the repo
1. Install npm if you haven't:
    ```bash
    sudo apt install npm
    ```
1. Install dependencies:
    ```bash
    npm install
    ```
1. Run the development server:
    ```bash
    npm run dev
    ```
