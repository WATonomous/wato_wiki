# WATcloud Quest Book - Summer 2025

## Term Objectives Summary

These are WATcloud's objectives for the Summer 2025 term. 

1. Hardware additions: nebula machines and wato-drive3
2. Handle outages, optimizing for zero downtime
3. Deploy more projects!
3. Write blog posts!
4. Get more funding!
