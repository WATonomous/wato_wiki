# WATcloud Quest Book - Summer 2025

## Term Objectives Summary

These are WATcloud's objectives for the Summer 2025 term. 

1. Hardware additions: nebula machines and wato-drive3
2. Handle outages, optimizing for zero downtime
3. Transition to login node architecture
4. Write blog posts!
5. Get more funding!

### Term Objectives and Scoring
1. **Hardware additions: nebula machines and wato-drive3**
| Score | Criteria                                                                 |
|-------|--------------------------------------------------------------------------|
| 20/20 | All 3 machines are in the cluster along with a new wato-drive            |
| 10/20 | All 3 machines is in the cluster                                         |

| Score | Criteria                                                                 |
